❓ Waarom werken emails niet?

🚫 Hoofdprobleem: EmailJS niet geconfigureerd

❌
Template IDs zijn placeholders
YOUR_TEMPLATE_ID, YOUR_2FA_TEMPLATE_ID zijn nog niet vervangen
❌
Service ID is test versie
service_doc2eqt is een placeholder, niet jouw Gmail service
❌
Public Key mogelijk verouderd
902hWQUoBZdWzwVfH moet vervangen worden door jouw key

✅ Oplossingen

🚀 Optie 1: Super Snelle Fix (5 minuten)

  1. Ga naar Quick Gmail Setup
  2. Vul je EmailJS gegevens in
  3. Kopieer de gegenereerde code
  4. Vervang in index.html
  5. Save & push → Emails werken!

📋 Optie 2: Volledige Setup (15 minuten)

  1. Volg de Gmail Setup Guide
  2. Maak EmailJS account + Gmail service
  3. Configureer alle 3 templates
  4. Update je website code
  5. Test alle email types

⚡ Optie 3: Tijdelijke Activatie (Browser Console)

Als je al EmailJS hebt geconfigureerd, gebruik deze console command:

enableEmailJS("YOUR_PUBLIC_KEY", "YOUR_SERVICE_ID", "template_verification", "template_2fa", "template_booking")

💡 FAQ

Q: Ik krijg wel codes, maar geen emails - is dat normaal?

✅ Ja, dat is precies zoals het hoort! De website toont codes in alerts omdat EmailJS nog niet is geconfigureerd. Je kunt alle functionaliteit gebruiken zonder echte emails.

Q: Hoeveel kost EmailJS?

✅ Gratis! 200 emails per maand gratis. Perfect voor een cinema website.

Q: Moet ik een Gmail account maken?

✅ Gebruik je bestaande Gmail! Je hoeft geen nieuw account aan te maken.

Q: Is dit veilig?

✅ Zeer veilig! EmailJS gebruikt App Passwords, niet je echte Gmail wachtwoord.
